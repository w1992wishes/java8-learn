# java 8 新特性学习笔记

java 9即将发布，在这之前，还是先把java 8 的新特性总结一下。

#### java 8 的十大特性

* **Lambda表达式**

* **接口的默认方法与静态方法**

* **方法引用**

* **重复注解**

* **扩展注解的支持**

* **Optional**

* **Stream**

* **Date/Time API \(JSR 310\)**

* **JavaScript引擎Nashorn**

* **Base64**

这些新特性中Lambda表达式和、Stream、Optional应该是用处最大，也最方便的特性。
